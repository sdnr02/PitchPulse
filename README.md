# PitchPulse - Cricket Tournament Management Platform

PitchPulse is a comprehensive SaaS platform built with Python FastAPI that enables cricket tournament organizers to manage multi-tenant tournaments with real-time WebSocket-powered live scoring, team registration, player management, automated match scheduling, event-sourced scorekeeping, integrated Stripe billing for subscriptions and entry fees, and responsive web interfaces for scorers, teams, and spectators.
